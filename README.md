# OPC Daily Signal

OPC（一人公司）每日决策情报。不是新闻摘要，是经过第一性原理过滤的可执行信号。

🤖 本仓库由 [Hermes Agent](https://github.com/NousResearch/hermes-agent) 自动生成并执行——从信息源扫描、信号筛选、深度分析到 Git 提交与 Telegram 推送，全链路 AI 驱动，零人工编辑。

## 核心公式

```
OPC存活 = (单价 × 数量) / 投入时间 > 成本
```

每条信号必须作用于公式中的某个变量，否则是噪音。

## 信号分类

| Emoji | 类型 | 作用变量 |
|-------|------|----------|
| 💰 | 提价信号 | 单价 ↑ |
| 📈 | 放量信号 | 数量 ↑ |
| ⚡ | 提效信号 | 时间 ↓ / 成本 ↓ |
| 💀 | 死亡信号 | 避坑 |
| 🆕 | 新发现源 | 情报网络扩展 |

## 三道铁律筛选

1. **作用于哪个变量？** — 标不出来的扔
2. **可复制吗？** — 依赖个人光环/运气/特殊资源的扔
3. **明天能改变一个动作吗？** — 不能的扔
