## 💰 单价｜Skills 作为可打包资产已经成市场

### → 事实
GitHub Trending（weekly）头位是 `mattpocock/skills`，一周 +31,091 stars，副标题"Skills for Real Engineers. Straight from my .claude directory."——把私人 Claude 目录搬上公开 repo，README 即引爆。同一周 `ComposioHQ/awesome-codex-skills` +3,964 stars，HN 当日 #22 是 Addy Osmani 的《Agent Skills》blog（69 pts）。三条独立信号在同一周指向同一概念：**skills（即结构化的 prompt + 工作流 + 脚本包）正在成为可独立销售的 AI agent 时代资产单元。**

### 底层逻辑
- LLM 能力高度同质后，差异化转移到「调用 LLM 的脚手架」。
- 传统 SaaS 卖"产品访问权"，skills 卖"做一件事的最佳实践打包"——交付物是 markdown + 脚本，无 hosting 成本，毛利接近 100%。
- 个人专家的隐性知识第一次有了边际成本为 0 的封装方式。

### 为什么 OPC 要关注
对一人公司而言，skills 是**公式中"单价"维度最理想的杠杆**：
- 单价：$99-499 一份打包，可加 $200/h 答疑 upsell；
- 数量：可在 Gumroad / Lemon Squeezy / GitHub 自动分发；
- 时间：录制一次 + 持续维护，生产函数次线性；
- 成本：无库存、无服务器。

### 🔑 行动
本周把过去 90 天最稳的 3-5 个 agent 工作流（codemod、文档同步、code review、release notes、客户 onboarding 邮件等）整成 skills bundle：免费版上 GitHub 引流，付费版 $99-299 卖配套（脚本 + 视频 + 1 次答疑）。

### 🔮 二阶 + 时间
6-12 个月内 Anthropic / OpenAI / Cursor 必出官方 skills marketplace（参考 GPT Store 的演化路径）。早一步占住 SEO 关键词与品牌位的人吃头部流量；marketplace 上线后无名气投稿即沉底。所以"今天-2026 年底"是窗口期。

### ⚠️ 反向检验
拿掉个人光环（X 粉丝、前同事推荐）后没人付费，说明卖的是关注度不是工作流——立刻砍价、改卖"为某个具体岗位定制 skills"的咨询时（$200/h）。

### 连锁效应
- 优质 skills 反哺个人品牌 → 演讲/咨询单价上浮；
- skills 沉淀的 prompt 资产可二次封装为 SaaS（同一资产复用）；
- 大厂 marketplace 抽成（预估 20-30%）会让早期独立分发者拥有定价权优势。

🔗 https://github.com/mattpocock/skills

## ⚡ 时间｜Codemod-as-a-Service 缝隙被 Stripe 实锤

### → 事实
Stripe 工程团队公开《Formatting an entire 25-million-line codebase overnight: the rubyfmt story》，详述一夜对 2500 万行 Ruby 跑 formatter 的工程取舍：并行化、回滚、CI 切片、reviewer 疲劳处理。HN 107 分。

### 底层逻辑
连 Stripe 这种工程 bench 顶级的公司都把"大规模 codemod"当独立项目立项写文章——意味着腰部 Series B-D 公司根本啃不动；过去这是高级架构师的隐性技能，现在 AST + Claude Code + Cursor 把单人执行变可行。

### 为什么 OPC 要关注
直接撬动公式中的"单价"+"时间"：
- 一次性服务 $5-25k，3-7 天交付；
- 不需要长期客户关系；
- 工具（ast-grep, comby, semgrep, Claude Code）已经把执行成本压到一人可承担。

### 🔑 行动
本周做"48h 全量 codemod 落地"landing page，定价分层 $5k（<100k 行）/ $25k（<2M 行）/ 报价制（>2M）。冷邮 50 家用 Ruby / Python / JS 的 Series B-D 公司 CTO，附 Stripe 这篇文章作为社会证明 + 你过去做过的一次小型 codemod 案例。

### 🔮 二阶 + 时间
12 个月内 Cursor / Claude 会推出"repo-wide refactor"原生命令，单价从 $25k 跌到 $500。窗口期 = 现在到 2027 中。在窗口内攒 5-10 个 case study，等工具普及后转型卖"codemod 流程培训 + 验收 SOP"，单价回到 $5-15k。

### ⚠️ 反向检验
前 20 封冷邮没拿到 1 个 30 分钟通话，说明买家不是 CTO 是平台工程经理——切换收件人画像、把 pitch 从"省时间"改成"消除技术债季度 OKR 风险"再试。

### 数据交叉验证
- HN 同周 #4 "I am worried about Bun"（383 pts）与 Stripe rubyfmt 同时高位，说明语言/runtime 切换是 2026 季度热议，迁移类项目刚需上升。
- GitHub Trending 周榜里 `abhigyanpatwari/GitNexus`（+4,694）是"客户端 code 知识图谱"，进一步降低 codemod 前期分析成本。

🔗 https://stripe.dev/blog/formatting-an-entire-25-million-line-codebase-overnight-the-rubyfmt-story

## 📈 数量｜强制开放数据 = OPC 分析层最干净的金矿

### → 事实
`FuelInsight` (UK) 上 HN 157 pts：一个人基于英国 CMA "Fuel Finder scheme"（强制 8000+ 加油站每日报价的合规披露数据）做的分析站，提供高速公路溢价、超市折扣、品牌对比、邮件订阅。底层数据政府免费开放，作者只做聚合 + 可视化 + SEO + 订阅。

### 底层逻辑
监管强制披露的数据具备三个稀缺特性：
1. **持续更新且免费**——零边际数据成本；
2. **结构化且权威**——天然 SEO trust signal；
3. **大公司不屑做、初创看不上**——竞争稀薄。

### 为什么 OPC 要关注
公式中"数量"的低成本放大器。一人独立运营，靠 SEO + 订阅，可在 6-9 个月内做到 $3-5k MRR 而无需融资。

### 🔑 行动
今天列出所在国家/行业里 3 个"强制披露但没人做体面前端"的数据源——美国 SEC EDGAR 私募 D 表、FDA 召回、各州 occupational license 库、欧盟 ESG CSRD、中国信用中国处罚公告、招投标。挑 1 个，48 小时内做 MVP（一张行业排行 + 邮件周报订阅 $9/月），用 Cloudflare Pages 零成本上线。

### 🔮 二阶 + 时间
3-6 个月内会有同质 LLM 包装站涌入，护城河不在数据（人人能爬）而在「细分客群的呈现方式」+「持续 ETL 不挂」。提前签 2-3 个行业 KOL 做内容分发合作，把 SEO 流量换成订阅，9 个月内可达 $3-5k MRR。

### ⚠️ 反向检验
上线 30 天 organic 流量 <500 UV，说明这个数据池没人关心——立刻换数据源，不要给自己加"内容/SEO 优化"动作，是选品错了。

### 连锁效应
积累的数据 ETL pipeline + SEO 资产可在 18-24 个月内打包出售（参考 SimilarWeb / OpenCorporates 早期路径），或转型为 API 服务（数量维度二次套利）。

🔗 https://www.fuelinsight.co.uk

## 💀 风险｜Horizontal AI 客服赛道封顶，OPC 不要正面打

### → 事实
Sierra（Bret Taylor 的公司）官宣 $950M 融资 / $15B 估值。HN 84 pts / 108 comments（评论多于投票通常意味争议）。

### 底层逻辑
"通用 AI customer experience"赛道资本面已结。Sierra 这种 ARR / 估值结构必然只服务 mid-market+（年合同 $100k+），下方留出"<$2k/月、垂直行业、需要本地数据隔离"的整片空白。

### 为什么 OPC 要关注
正面打 Sierra = 死。但其估值天花板恰好暴露下方"垂直长尾 + 实施咨询"的真空。OPC 应做"陪跑式实施"而非"平台"。

### 🔑 行动
本周锁定 1 个垂直（牙科诊所 / 独立律所 / 精品民宿 / 驾校），做"每月 $199-499，含 100 通电话/邮件回复 + 行业话术模版 + 本地化部署"的极轻 AI 客服，用 OpenAI Realtime + n8n 跑通。不要做平台，做实施。

### 🔮 二阶 + 时间
18-24 个月内 Sierra / Decagon 会 down-market 抢 SMB。那时你已握住 30-50 个垂直客户的真实工单数据——可做成行业 fine-tune 模型卖回给 Sierra 或被收购。

### ⚠️ 反向检验
3 个月签不到 5 个付费客户，说明所选垂直没有"客服 = 收入"的结构（如 ToB 长销售周期客户根本不在乎客服自动化）——换成"漏一通电话 = 丢一单"的高频低价垂直再试（汽修、宠物医院、外卖加盟）。

### 数据交叉验证
HN 当日 #7 "How OpenAI delivers low-latency voice AI at scale"（239 pts）+ Sierra 融资同周——基础设施层（OpenAI Realtime）+ 应用层（Sierra）双重官方背书，垂直实施层确认存在。

🔗 https://sierra.ai/blog/better-customer-experiences-built-on-sierra
