## 💰 单价 | Skill 包成为新一代知识产品

→ **事实**：mattpocock/skills 一周 18,278 stars，addyosmani/agent-skills 7,841 stars，均为 .claude 目录直接开源的"工程师 Skill 包"。Skill = 触发条件 + 步骤 + 陷阱的结构化 markdown，被 Claude Code / Codex / Cursor 等 agent 直接挂载执行。两个仓库同周冲到 trending 前列，说明"Skill 即产品"形态已被市场验证。

🔑 **动作**：把自己最常用的 3 个工程 workflow（如"PR review"、"npm 包发版"、"Stripe 接入"）按 SKILL.md 格式写出，打包成 Gumroad 收费包（$29-99），或挂上 GitHub 做 lead-gen 接咨询。

🔮 **二阶（6 个月窗口）**：会出现 Skill marketplace（npm + Gumroad 混合形态），头部作者通过 skill 订阅变现。先发优势窗口约 3 个月。

⚠️ **反向检验**：如果你的 skill 只是"prompt 模板"而无"陷阱/验证"部分，会被 agent 自带的 system prompt 替代，付费意愿归零。差异化必须在踩坑细节。

🔗 https://github.com/mattpocock/skills

### 底层逻辑
- 知识产品演化路径：书籍 → 视频课程 → 代码模板 → SaaS → **agent 可执行的过程性知识**。每一代都把"用户从输入到产出的距离"压短。
- Skill 包是第一种"零集成成本"的知识产品：买家不需要复制粘贴、不需要看完，丢进 ~/.claude 目录即用。
- 购买决策从"我有时间学吗"变成"省 1 小时值不值 $29"——后者几乎必买。

### 为什么 OPC 要关注
- 写一次卖 N 次，库存为零，物流为零。**OPC 最理想的产品形态**。
- Skill 写作成本对 senior 开发者趋近于零：你已经踩过坑，只需要把过程写下来。
- 分发渠道现成：GitHub 关注度（lead-gen）+ Gumroad（付费转化）+ X 帖子（流量）。

### 连锁效应
- Anthropic 大概率会推出官方 skill registry（半年内）。早期作者会被默认纳入推荐名单。
- 出现 skill-as-a-service：企业按席位订阅整个 skill 库（类似 1Password Teams）。
- 反向影响开发者招聘：履历上写"开源 skill 100k+ downloads"会成为新 credential。

### 数据交叉验证
- addyosmani（Google Chrome team）入场，说明这不是 indie 圈自嗨，主流大厂工程师认可。
- 同周 anthropics/financial-services 9,480 stars——Anthropic 官方在用同样的 skill 形态做行业模板。

## ⚡ 时间 | Stealth Browser 解锁数据采集业务

→ **事实**：CloakHQ/CloakBrowser 一周 9,120 stars，"30/30 bot detection 测试全过"，Playwright 的 drop-in 替代，源码级 fingerprint patch。过去需要 Bright Data / ScrapingBee 每月 $500+ 才能跑通的反爬场景（Cloudflare、Akamai、DataDome），现在单机本地脚本就能搞定。

🔑 **动作**：本周在本地起一个 CloakBrowser，挑一个垂直数据集（如"国内 SaaS 定价页快照"、"招聘 JD 技能词频"），做成 weekly digest，订阅 $19/月或一次性数据集 $99。

🔮 **二阶（3-6 个月窗口）**：反检测进入军备竞赛——Cloudflare 大概率推出针对此类指纹补丁的新一代检测。窗口期需要尽快锁定客户和数据资产。

⚠️ **反向检验**：如果项目目标客户是合规敏感型（金融、医疗），任何"绕过反爬"标签都是毒药，反而把单价打下去。To-B 慎用。

🔗 https://github.com/CloakHQ/CloakBrowser

### 底层逻辑
- 数据采集成本结构 = 代理 IP（$200-2000/月）+ stealth 维护（人力 20-40h/月）+ 计算（$50-200/月）。
- CloakBrowser 把"stealth 维护"压到接近 0，砍掉最大的变动成本。
- 释放的时间重新投入"数据加工 + 客户开发"——价值更高的环节。

### 为什么 OPC 要关注
- 数据业务客单价高、复购率高、单兵作战可行。**典型 OPC 友好赛道**。
- 不需要找 PMF，只需要找到"愿意付钱、不会自己抓"的垂直人群。
- 数据资产可累积——今天抓的数据 12 个月后还能卖（历史价格、招聘趋势）。

### 连锁效应
- 数据 broker 中间商（Bright Data 等）承压，估值下行。
- 反爬技术供应商（DataDome、PerimeterX）推出"AI-resistant"产品，价格上涨 50-100%。
- 国内出现"个人数据 dealer"小型作坊，做 B 端定制数据包。

## 📈 数量 | 免费 AI 编程路由器降低试错成本

→ **事实**：decolua/9router 一周 5,377 stars，"unlimited FREE AI coding"，把 Claude Code/Codex/Cursor/Cline/Copilot 路由到 40+ 免费 provider，自动 fallback，RTK -40% tokens。配合 Hmbown/DeepSeek-TUI（8,701 stars）。Solo dev 端 AI 编程边际成本归零。

🔑 **动作**：把 side project 的 coding agent 切到 9router + DeepSeek-TUI 组合，每月省下的 $100-300 转为投流/广告预算。

🔮 **二阶（12 个月窗口）**：OpenAI/Anthropic 大概率封禁聚合 provider 或加 IP 风控。短期红利明确，生产系统不要绑死在 router 上。

⚠️ **反向检验**：付费客户的生产代码，质量波动和 provider 串味引发隐藏 bug，省的 $100 可能赔 $1000 客户信任。仅用于实验和 PoC。

🔗 https://github.com/decolua/9router

### 底层逻辑
- AI 编程成本曲线：2023 GPT-4 月均 $200+ → 2025 Claude Code $50-150 → 2026 router 时代 $0-20。
- 边际成本归零的直接后果：试错次数可以乘以 5-10 倍。OPC 公式中"数量"的乘数被打开。
- 同时降低"放弃成本"——一个项目跑两周没起色直接停，不再有沉没成本焦虑。

### 为什么 OPC 要关注
- Solo 最稀缺的不是钱，是"敢于砍掉的勇气"。AI 编程免费让砍项目更容易。
- 并行 3-5 个实验项目，等其中 1 个出现 traction 再 all-in，**用数量换确定性**。
- 省下来的预算可以买流量（X ads、Reddit 投放），把"造产品成本"和"获客成本"重新平衡。

### 连锁效应
- 中小型 AI coding SaaS（Bolt、v0、Lovable）的免费层会变薄，竞争压力增加。
- OpenAI 推出"开发者 cap"政策的可能性上升。
- 中国 DeepSeek/Qwen 系生态在海外 indie 圈渗透率快速提升。

## 💀 反向陷阱 | 警惕"AI Psychosis"型公司

→ **事实**：HN #1 热帖（681 分，304 评论）转述 Mitchell Hashimoto 观察："I believe there are entire companies right now under AI psychosis"——被自家 AI demo 催眠、忽视 unit economics、靠融资延寿的团队。同日 Astral Codex Ten《The sigmoids won't save you》（136 分）数学层面论证 AI 能力曲线不会持续指数。

🔑 **动作**：把每个项目过审："这个功能在 GPT-3.5 时代是否成立？" 不成立的（依赖"再过一代模型就能 work"的假设）立即砍掉或转型。OPC 不能押注模型进步。

🔮 **二阶（6-12 个月窗口）**：AI startup 估值出现分层崩塌——拿不出"模型 zero-progress 也能跑"现金流的公司融不到下一轮。Solo 反而占优：没有 burn rate 就没有死线。

⚠️ **反向检验**：如果产品已经有付费 retention 且不靠模型新能力推新功能，本条不适用，反而是收购 distressed AI 公司用户的机会。

🔗 https://news.ycombinator.com/item?id=48153379

### 底层逻辑
- "AI Psychosis"本质是 demo-to-product gap：80 分的 demo 在融资 ppt 完美，production 需要 99.5 分，差 19.5 分是 5-10 倍工程量。
- VC 因为 FOMO 把 demo 当 product 投，市场被"伪 product"塞满——一旦模型进步放缓，伪 product 集体失血。
- Hashimoto 是 Hashicorp 创始人（$5B IPO），可信度远高于一般 X 段子手。

### 为什么 OPC 要关注
- Solo 没有 VC 钱可烧，**被迫从第一天就 ship product 而非 ship demo**——结构性优势。
- 当大公司因 burn rate 收缩时，OPC 可以低价收购 distressed 项目的用户、域名、品牌。
- 不被 AI 叙事绑架的产品（CRUD SaaS、内容订阅、教育）反而成为反脆弱选择。

### 连锁效应
- 2026 下半年开始一批 AI startup 进入 down round 或被收购，二级人才流入市场。
- VC 转向 "AI-applied"（应用层）而非 "AI-native"（模型层）赛道，对 indie 友好。
- 大量 AI agent / wrapper 类项目消亡，留下来的赛道形成稳定格局。

### 数据交叉验证
- Astral Codex Ten 同期文章"sigmoids"独立从数学侧得出相同结论。
- HN 评论区 304 条讨论中，多名一线创始人匿名承认自己公司处于"AI psychosis 状态"。
